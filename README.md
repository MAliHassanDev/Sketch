## Get Started
Follow these steps to run Whiteboard locally:
1. **Clone the repository**:
```bash
  git clone https://github.com/MAliHassanDev/WhiteBoard.git
```
2. **Navigate to the project directory**: 
```bash
    cd WhiteBoard
```
3. **Install dependencies**:
```bash
    npm install
```
4. **Run dev server**:
```bash
    npm run dev
```
